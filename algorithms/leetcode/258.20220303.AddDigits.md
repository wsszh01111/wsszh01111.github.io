# [258. 各位相加](https://leetcode-cn.com/problems/add-digits/)

## 思路1

模拟即可 不过时间比较慢

```js
function addDigits(num) {

  function count(num) {
    let res = 0
    while(num) {
      res += num % 10
      num = Math.floor(num / 10)
    }
    return res
  }
  
  let res = count(num)
  while(res > 9){
    res = count(res)
  }
  return res
}
```
